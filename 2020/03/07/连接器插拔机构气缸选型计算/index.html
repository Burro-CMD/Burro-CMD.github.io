    连接器插拔机构气缸选型计算

    Posted on 2020-03-07 In Tech Views: Word count in article: 2.8k Reading time ≈ 3 mins.
    连接器插拔机构气缸的选型。

    需求

      为一款USBC receptacle的plug模组气缸选型，以型号217B-CA05的C口连接器为例，Datasheet，插拔力：

    数据分析

    对气缸来说，需要完成两个方向的力：

    动作 连接器给出的力 气缸需要克服
    插入 5 ~ 20 N 最大插入力 + 机构摩擦 + 安全系数
    拔出 8 ~ 20 N 最大拔出力 + 机构摩擦 + 安全系数

    所以基础计算按最大值取：

    • F_insert_max = 20N
    • Fextract_max = 20N

    注意：
    20 N 是连接器插拔力，不是建议让气缸直接输出很大的力。
    气缸理论力如果太大，可能会压坏 USB‑C receptacle、PCB 焊点或 plug 舌片，所以后面要同时考虑：

    • 气缸必须有足够能力完成插拔；
    • 实际作用到连接器上的力必须被限制在合理范围内。

    模组阻力

    Plug模组一般包括：

    • 气缸
    • 线性导轨
    • Plug固定机构
    • 浮动接头
    • 限位结构
    • 可能还有弹性缓冲结构

    插拔方向若是水平，重力基本不参与轴向力；如果是垂直方向，需要加上模组重量。

    1. 一般工程估算：
      F_friction = 3 ~ 8 N

    2. 如果导轨质量较好、滑块较小，可以先取：
      F_friction = 5 N

    1. 所以单方向基础需求力为：
      F_base = F_connector_max + F_friction = 20 + 5 = 25 N

    2. 安全系数

    气缸选型时，不能刚好等于理论需求，因为实际存在：

    • 气压波动
    • 密封摩擦
    • 滑台摩擦变化
    • plug 与 receptacle 的同轴度误差
    • 机构老化
    • 温度影响
    • 动态冲击

    建议安全系数：S = 2 ~ 3

    测试治具、自动化插拔机构，建议取：S = 2.5

    则气缸有效输出力要求：
    F_required = F_base × S = 25 × 2.5 = 62.5 N

    所以气缸在插入和拔出方向，实际有效力最好都能达到约：
    F_required ≈ 60 ~ 65 N

    气缸输出力公式

    双作用气缸有两个方向作用力。

    伸出推力

    F_push = P × A_piston × η，其中 A_piston = πD² / 4

    缩回拉力

    F_pull = P × A_annular × η，其中 A_annular = π ×, D² - d², / 4

    符号 含义
    P 工作压力，单位 MPa，1 MPa = 1 N/mm²
    D 气缸缸径，mm
    d 活塞杆直径，mm
    η 气缸效率，一般取 0.7 ~ 0.85
    F_push 伸出推力
    F_pull 缩回拉力

    一般工程计算可先取：
    P = 0.5 MPa, η = 0.75

    缸径力值估算

    1. 工作气压：P = 0.5 MPa

    2. 气缸机械效率：η = 0.75

    3. 气缸类型：标准双作用气缸

    4. 活塞杆直径按常规比例选取（经验值）：
      φ8 → d=4

    • 活塞面积

    A_p = πD² / 4

    • 有效推力（伸出）

    F_push = P × A_p × η

    • 有效拉力（缩回）

    F_pull = P × A_r × η
    A_r = π(D² - d²) / 4

    • φ8气缸

    • 面积

    A_p = π × 8² / 4 = π × 64 / 4 = 50.27 mm²

    • 推力

    F_push = 0.5 × 50.27 × 0.75 ≈ 18.8 N

    • 拉力（d=4）

    A_r = π(64 - 16)/4 = π×48/4 = 37.7
    F_pull = 0.5 × 37.7 × 0.75 ≈ 14.1 N

    • 以此类推计算所得：

    缸径 有效推力 有效拉力 是否满足约 62.5 N
    φ8约 18.8 N 约 14.1 N 不满足
    φ10约 29.5 N 约 24.7 N 不满足
    φ12约 42.4 N 约 31.8 N 不满足
    φ16约 75.4 N 约 64.8 N 满足
    φ20约 117.8 N 约 99.0 N 满足，但偏大

    • 所以如果按：

    • 最大插拔力 20 N

    • 机构摩擦 5 N

    • 安全系数 2.5

    • 工作压力 0.5 MPa

    • 效率 0.75

    • 来选，推荐最小缸径约为：

    φ16 双作用气缸

    注意

      虽然按安全系数计算，φ16 气缸比较合适，但 USB‑C receptacle 本身插拔力只有 20 N 级别。如果 φ16 气缸在 0.5 MPa 下直接硬推，实际有效推力约 75 N，可能造成：

    • receptacle 塑胶舌片损伤；
    • plug 端子变形；
    • PCB 焊盘受力过大；
    • 连接器外壳变形；
    • 插偏时直接撞坏产品。

    所以实际设计中要做力限制。

    气缸方案

    方案 A：φ16 双作用气缸 + 调压阀 + 浮动接头，推荐

    气缸选

    • 双作用气缸
    • 缸径：φ16
    • 行程：根据 plug 插入深度确定，通常 10 ~ 20 mm
    • 工作压力：0.15 ~ 0.25 MPa 调试

    φ16 气缸面积约：

    A = π × 16² / 4 ≈ 201 mm²

    若按效率 η = 0.75，压力 0.18 MPa 时：

    F_push = 0.18 × 201 × 0.75 ≈ 27 N

    缩回拉力约：

    F_pull ≈ 23 N

    这样实际力比较接近 USB‑C 的插拔力需求，调试空间也比较好。

    建议实际调试压力：

    0.15 ~ 0.20 MPa 起调

    如果拔出不稳定，再逐步增加到：

    0.20 ~ 0.25 MPa

    行程选择

     USB‑C plug 的有效插入深度通常只有几 mm，但自动化机构不能只按插入深度选行程，还要考虑：

    • plug 初始避让位置；
    • 产品放置误差；
    • 预压缓冲距离；
    • 传感器触发距离；
    • 机构限位空间。

    建议：

    气缸行程 = 实际插入深度 + 安全退让距离 + 调试余量

    • USB-C 插入有效行程：约 6 ~ 8 mm
    • 退让距离：3 ~ 5 mm
    • 调试余量：2 ~ 5 mm

    所以：

    行程 10 mm，适合空间紧凑场合
    行程 15 mm，推荐
    行程 20 mm，调试余量大

    推荐优先考虑：

    φ16 × 15 mm 双作用气缸

    推荐配置

    项目 推荐
    气缸类型 双作用气缸
    缸 径φ16
    行程 15 mm 或 20 mm
    工作压力 0.15 ~ 0.25 MPa 调试
    最大供气 可按 0.5 MPa 设计，但需调压
    导向方式 外置线性导轨或滑台
    接头 浮动接头
    速度控制 双向节流阀，排气节流
    检测 前后磁性开关
    保护 机械限位 + 弹性缓冲
    可选 小量程力传感器，0 ~ 50 N 或 0 ~ 100 N
