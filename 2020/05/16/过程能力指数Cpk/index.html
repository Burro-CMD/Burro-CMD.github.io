过程能力指数Cpk

Posted on 2020-05-16 In Tech Views: Word count in article: 1k Reading time ≈ 1 mins.
过程能力指数概念，计算公式以及与Cp的不同。

简介

  1. Cp（Process Capability Index，过程能力指数）：衡量的是过程在没有考虑均值偏移的情况下，其分布范围相对于规格限的宽窄程度。它反映了过程的潜在能力或“理想”状态下的能力。
  2. CpkProcess Capability Index with K, 带有偏移量的过程能力指数。它衡量的是你的工艺输出相对于规格限的稳定程度。

计算公式

  1. Cp = (USL-LSL)/6σ= T/6σ
    T=允许最大值（USL）-允许最小值（LSL）

  2. Cpk = min[(USL – μ) / (3σ), (μ – LSL) / (3σ)]

    其中：
    USL = 规格上限（Upper Specification Limit）
    LSL = 规格下限（Lower Specification Limit）
    μ = 过程均值
    σ = 过程标准差

解读

  1. Cp的意义：

  • 主要用于评估过程的潜在能力，即如果过程均值恰好位于规格中心时，过程能够满足规格要求的能力。

  • 对于新设计的过程或经过重大改进后的过程，Cp是一个有用的指标，因为它不考虑历史数据中的均值偏移。

  1. Cpk的意义:

  • 更贴近实际情况，因为大多数过程都会存在一定的均值偏移。

  • 用于评估当前过程在满足规格要求方面的实际能力。

  • 在持续改进过程中，Cpk是衡量过程性能改善的重要指标。

  1. CPK＜1.0 制程不良多，必须整改
  2. 1.0≤CPK＜1.33 一般水准，需改善
  3. 1.33≤CPK＜1.67 良好，稳定可控
  4. 1.67≤CPK＜2.0 优秀，高精量产
  5. CPK≥2.0 极高精度，冗余充足

简单说：Cpk 越大，说明你的工艺越稳定，产品越不容易超出规格。

比较与选择

  • 当过程处于稳定状态且均值接近目标值时，Cp和Cpk的值会相近。然而，在实际应用中，由于各种因素的影响（如原材料变化、设备磨损等），过程均值往往会发生偏移，此时Cpk更能准确反映过程的实际能力。

  • 选择使用哪个指标取决于具体的应用场景和目标。对于新项目或过程的初步设计阶段，可能更关注Cp以了解潜在能力；而对于现有过程的监控和改进，则更倾向于使用Cpk来评估实际性能和进行决策。

延申

Ppk（过程性能指数）和 Cpk 的计算公式相同，区别在于标准差的计算方式：

  • Cpk 使用的是组内标准差（within-group），反映短期能力
  • Ppk 使用的是总体标准差（overall），反映长期性能

一般来说 Ppk ≤ Cpk。如果两者差距很大，说明过程中存在特殊原因变异（比如设备漂移）。
