FCT治具机械结构与设计规范

Posted on 2020-01-20 In Tech Views: Word count in article: 3.8k Reading time ≈ 3 mins.
功能性测试治具FCT的结构设计与一些常规的设计规范。

FCT概述

  功能测试 (FCT) 夹具是一种专用测试系统，旨在模拟实际工况，验证印刷电路板组件 (PCBA) 或成品电子产品的运行性能。与结构或元件级测试不同，FCT 评估产品是否按照预期设计运行。

FCT治具结构

下压系统 (Cylinder / Toggle Clamp)

  压紧系统是功能测试 (FCT) 夹具中的关键机械子系统。其主要作用是施加可控且可重复的力，以确保测试探针（弹簧探针）与被测单元 (UUT)（通常为印刷电路板组件 (PCBA)）之间可靠的电接触。

压板 (Acrylic / FR4 Plate)

  压板是功能测试 (FCT) 夹具中的关键机械部件。它位于压机系统和被测单元 (UUT)（通常为印刷电路板组件 (PCBA)）之间，负责在测试周期内将施加的力均匀分布到电路板上。

UUT (PCBA) Unit Under Test

  被测单元 (UUT) 是指在 FCT 夹具中被测试的设备，通常是印刷电路板组件 (PCBA) 或完全组装的电子产品。

探针 (Pogo Pins)

  测试探针广泛应用于半导体（IC）及电路板（PCB）测试领域，探针被安装在 插座或夹具内部，插座或夹具安置在被测物和测试基板的中间，探针连接被测物 与测试基板，通过传输一定电流和频率来判定被测物是否合格。

  • Double action probe　Movable plunger: Two

  • Single action probe　Movable plunger: One

探针板(FR4) + 导向板

  探针板组件是用于固定和对准测试探针的结构平台。它通常由一块FR4主板以及上下导向板组成，导向板也叫浮动板。

针板ESD要求

  • Material itself ESD: 1x105Ω<Resistance<1x109Ω
  • Material to ground: 0<Voltage<100V
  • Material to ground: 0<Resistance to ground<1Ω

转接板 (Wire Harness / PCB)

  转接板，也称为适配器板或接口PCB，是一种中间电路板，用于连接探针接口（弹簧针）和外部测试系统（ATE）。它在FCT夹具中充当结构化、工程化的信号路由层。

与传统的线束不同，接口板为信号分配提供了一种可控、可扩展且易于维护的解决方案。

转接头 (To Test Equipment)

  接口连接器用作夹具与外部测试设备（ATE 系统）之间的连接点。

测试系统 Power / DAQ / Software

  ATE系统是FCT装置的控制和测量核心。它执行测试序列，控制功率和信号，并评估被测设备的性能。

公差分析（Tolerance Stack-Up Analysis）

  公差分析是指评估所有影响探针与被测单元（UUT）对准的机械部件的累积尺寸偏差的过程。
其主要目标是在最不利的尺寸条件下确保探针与焊盘之间可靠的接触。

公差尺寸链

  • Guide System (Pins/ Bushings)——导向轴/轴承公差
  • Probe Plate Hole Position——针板孔位公差
  • Guide Plate Position——导向板位公差
  • UUT locating holes/ edges——被测物孔/ 边线公差
  • PCB fabrication tolerance——PCB板制造公差
  • Test pad position——测试点位置公差

最小距离和最小测试点尺寸

  每种测试夹具都存在不同的公差，导致测试点位置与最佳位置存在偏差。因此，测试点的最小尺寸以及测试点与被测电路板元件之间的最小距离都存在实际的限制。以下是一些典型值的概述：

距离定义

  • C: Centering hole diameter
  • D: Distance between test point (edge) and centering hole (edge)

应力分析（Simulation）

  应力分析旨在评估操作过程中被测单元（UUT）所承受的机械载荷和由此产生的应力。一般是±500微应变。
